== Description ==

Designless
Smart layout with simple UI and common UX patterns

Conversational
Distraction-free form experience

Multistep
Group your questions in a multistep form

Simple editor
Gutenberg-based form builder

Asyncronous
Async data submit with WP REST API

== Translations ==

Italian translation included

== Installation ==

1. Upload the entire `formality` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Formality** menu in your WordPress admin screen.
